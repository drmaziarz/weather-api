Weather Data API

Use the following URLs to access weather data:

  • http://127.0.0.1:5000/api/v1/station/date - get the temperature for a specific station on a specific date
  • http://127.0.0.1:5000/api/v1/station - get the temperature for a specific station for all dates
  • http://127.0.0.1:5000/api/v1/yearly/station/year - get the average temperature for a specific station for a specific year

Note: Replace "station" and "date/year" with the desired values.
